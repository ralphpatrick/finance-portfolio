How to Use This System

1

View Online

This site is already live on GitHub Pages. Simply navigate through the modules using the buttons above. All data loads dynamically from CSV files.

2

Run Locally

Clone the repository and open index.html in any modern browser. No build process required‚Äîit's pure HTML/CSS/JS.

git clone https://github.com/yourusername/finance-operating-system.git
cd finance-operating-system
open index.html
3

Deploy Your Own

Fork the repo, then enable GitHub Pages:

  1. Go to repository Settings
  2. Navigate to Pages section
  3. Select "main" branch as source
  4. Save and wait ~1 minute
  5. Your site will be live at: username.github.io/finance-operating-system
4

Customize Data

Edit any CSV file in the /data folder. Modules automatically reload with your changes. All calculations update dynamically.
